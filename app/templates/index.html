AgroSmart: Layanan Rekomendasi Tanaman

AgroSmart adalah sebuah sistem berbasis model pembelajaran mesin yang membantu pengguna menemukan tanaman terbaik untuk ditanam di lahan mereka. Model ini mengolah data faktor lingkungan seperti Nitrogen, Fosfor, Kalium, suhu, kelembaban, pH, dan curah hujan untuk memberikan rekomendasi yang paling sesuai.

Proses dimulai dengan pengguna memasukkan data lingkungan terkait ke dalam sistem. Data tersebut akan diproses melalui langkah-langkah pre-processing seperti pengkodean label, normalisasi data, dan pembagian dataset menjadi data pelatihan dan pengujian. Model kemudian dilatih menggunakan arsitektur Neural network dengan lapisan Dense yang dioptimalkan untuk klasifikasi multi-kelas.

Cara Kerja di AgroSmart:

  • Masukkan Data: Gunakan formulir untuk mengisi data lingkungan Anda, termasuk kandungan nutrisi tanah, parameter iklim, dan karakteristik pH.
  • Klik Cari Rekomendasi: Sistem secara otomatis memproses data melalui model machine learning yang telah terlatih.
  • Dapatkan Rekomendasi Tanaman: Hasilnya adalah tanaman paling cocok untuk kondisi tanah dan lingkungan Anda.
